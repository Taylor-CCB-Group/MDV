# Playwright Tests

Playwright tests live in this directory.

**Humans:** `docs/playwright/PLAYWRIGHT_GUIDE.md` — quickstart, running tests, writing tests, debugging, troubleshooting.

**Agents/LLMs:** `docs/playwright/PLAYWRIGHT_AGENT.md` — rules, commands, prompt templates.

Reference docs:

- `docs/playwright/PLAYWRIGHT_ARCHITECTURE.md` — fixtures, project lifecycle, execution policy
- `docs/playwright/PLAYWRIGHT_WRITING_TESTS.md` — how to add and extend tests
- `docs/playwright/PLAYWRIGHT_HISTORY.md` — background and prior failure modes
- `docs/playwright/PLAYWRIGHT_ROADMAP.md` — planned work (CI, coverage, speed, playwright-cli)

Directory layout:

- `catalog/`: route-mocked browser tests
- `project/`: backend-backed project tests
- `utils/`: shared helpers and project fixtures
- `test-data/`: checked-in test inputs
