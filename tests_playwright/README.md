# Playwright Tests

Playwright tests live in this directory.

- Current setup, commands, fixture architecture, suite status, and troubleshooting:
  `docs/PLAYWRIGHT_GUIDE.md`
- Historical background, migration notes, and prior failure modes:
  `docs/PLAYWRIGHT_HISTORY.md`

Directory layout:

- `catalog/`: route-mocked browser tests
- `project/`: backend-backed project tests
- `utils/`: shared helpers and project fixtures
- `test-data/`: checked-in test inputs
